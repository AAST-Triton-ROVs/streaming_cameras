# streaming usb cameras

# On rpi:

1- Enter vision directory: cd vision

2- activate venv: source venv/bin/activate

3- run: python3 usb_pi_send.py

# On station:

1- Make sure to clone the repo or create a folder containing Hybrid_Filter.py and Station_Receiver.py

2- run from inside the folder: python3 Station_Receiver.py


# Run on rpi then on station
# Make sure to use a static ip 192.168.1.60 on station
